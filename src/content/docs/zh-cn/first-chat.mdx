打开一篇长文章。在侧边栏里输入：

> 总结这个页面，给我五条要点。

阅读器用的是同一棵树），把它喂给模型，然后把答案流式输出到面板里。

打开一个表单（注册、问卷、设置页）然后告诉 Browy 你要填什么：

> 我的名字是 Alex Doe，邮箱是 alex@example.com。用这些信息填表单，

智能体按可访问性索引点击每个字段、输入值、在提交前停下来让你检查。

## "找一张最便宜的机票" <Badge text="多步" variant="note" />

打开 google.com：

> 搜一下 15 号 SFO 到 JFK 的机票，按价格排序，告诉我最便宜的那张。

智能体会导航、输入查询、点进结果页、读取里面的内容，然后回报给你。

## 工具栏

侧边栏标题栏（右上角）有四个控件：

| 按钮 | 作用 |
|---|---|
| **停止**（红色 ■） | 取消进行中的智能体回合。等同于 `Esc`。 |
| **新对话** | 把当前对话保存到历史，开一段新的。 |
| **历史** | 打开历史对话面板，可搜索的全部以往会话列表，可删除、可点击恢复。 |
| **设置** | 打开扩展的选项页（模型选择、[工具开关](/zh-cn/tools/#configurable-per-tool-toggles)、登录、诊断）。 |

下方的状态栏显示当前激活的浏览器、标签页标题，以及一个 **`N/M`**
徽标表示连接上 / 已安装的浏览器数。点击设置可以唤起一个已安装但
还没连上的浏览器。

## 输入

| 按键 | 动作 |
|---|---|
| `Enter` | 发送消息 |
| `Shift+Enter` | 在 prompt 里换行 |
| `Esc` | 取消进行中的回合 |

输入框会随着内容自动增高，支持多行输入，智能体回复里的 Markdown
（代码块、列表、链接、表格）会被正确渲染。

## 一轮回合里发生的事

- **流式文字**：模型边生成、面板边显示。
- **工具调用**：以行内卡片形式显示（`▸ click_index { index: 14 }`）。
  点击展开看参数和结果。失败的工具调用会带着错误信息一起显示。
- **推理过程**（如果模型输出了的话）会在答案上方用暗色块显示。
- **登录错误**直接显示一个 "Sign in to GitHub Copilot" 按钮，
  不需要进设置页找。

如果本地主机断开了，面板顶部会出现一个横幅提供一键重连
（如果主机没装的话也会告诉你怎么装）。

## 历史对话

每段对话都保存在 `chrome.storage.local`，带标题、预览和时间戳。
打开 **历史** 面板可以：

- 按标题或预览文本筛选
- 点击某一行恢复（智能体会重新绑定到那个会话 id）
- 点 × 删除一段对话
- **+ 新对话** 开一段新的

历史只属于你当前的浏览器配置，永远不离开你的机器。

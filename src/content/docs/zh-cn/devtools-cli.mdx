Browy 扩展会注册一个 **DevTools 面板**，里面是一个终端风格的智能体
REPL。在任何页面打开 Chrome DevTools，点 **Browy** 标签页，你就有
了和侧边栏一样的智能体，只是改成 CLI 而不是对话气泡。

## 斜杠命令

输入 `/` 触发自动补全。别名：`/?` → `/help`，`/models` → `/model`。

### 会话

| 命令 | 作用 |
|---|---|
| `/help` | 列出所有可用命令 |
| `/clear` | 清空滚动区（也可以用 `Ctrl+L`） |
| `/new` | 开启一个新对话 |
| `/reset` | 放弃当前会话，从头开始 |
| `/cancel` | 取消进行中的智能体回合（也可以用 `Esc`） |
| `/sessions` | 列出最近的对话会话 |
| `/copy` | 把上一条回复复制到剪贴板 |
| `/export` | 把滚动区导出成 `.txt` 文件 |
| `/history` | 显示最近的输入历史 |

### 模型与登录

| 命令 | 作用 |
|---|---|
| `/model` | 显示当前模型 + 打开实时选择器 |
| `/model <id>` | 直接切换模型，比如 `/model claude-haiku-4.5` |
| `/login` | 触发 GitHub Copilot 的 device-flow 登录 |

### 页面内省（不走智能体）

下面这些命令在 DevTools 本地执行，不会调用模型。

| 命令 | 作用 |
|---|---|
| `/url` | 打印当前被检查页面的 URL |
| `/title` | 打印当前页面的 `<title>` |
| `/tabs` | 列出打开的浏览器标签页（点击切换） |
| `/snapshot` | 页面快速摘要（标题、URL、各级标题） |
| `/dom <selector>` | 对 CSS 选择器做 `querySelectorAll` 摘要 |
| `/eval <expr>` | 在被检查页面里执行 JavaScript |
| `/js` | 切换 **JS REPL 模式**：每一行都在被检查页面里执行，直到再次 `/js` |

### UI 与元数据

| 命令 | 作用 |
|---|---|
| `/theme [auto\|dark\|light]` | 设置面板主题 |
| `/version` | 显示版本信息 |
| `/feedback` | 在 GitHub 上打开反馈页 |
| `/exit` | 关闭此面板的提示（真正关闭请关掉 DevTools） |

选择器目前可见的模型（取决于你的 Copilot 订阅）：

- Claude 系列：`claude-sonnet-4.6`、`claude-sonnet-4.5`、`claude-haiku-4.5`、
  `claude-opus-4.7`、`claude-opus-4.6`、`claude-opus-4.5`
- GPT 系列：`gpt-5.5`、`gpt-5.4`、`gpt-5.3-codex`、`gpt-5.2-codex`、
  `gpt-5.4-mini`、`gpt-5-mini`、`gpt-4.1`
- 以及 `auto`，让 Copilot 自己挑

工具栏里的模型徽章始终显示当前激活的选择。

## 键盘快捷键

| 按键 | 动作 |
|---|---|
| `Enter` | 发送 |
| `Shift+Enter` | 在 prompt 里换行 |
| `Esc` | 取消进行中的请求 |
| `Ctrl+L` | 清空日志 |
| `↑` / `↓` | 浏览历史（输入框为空时） |
| `/` 然后 `Tab` | 自动补全斜杠命令 |

## 什么时候用 CLI、什么时候用侧边栏？

| 如果你想... | 用... |
|---|---|
| 一个常驻在页面边上的对话，截图内联显示 | **侧边栏** |
| 留在 DevTools 里，针对被检查的目标快速发智能体提示 | **DevTools CLI** |
| 多个智能体并行，每个绑定一个被检查的标签页 | **DevTools CLI**（在每个标签页里打开 DevTools） |
| 和智能体结对编程实现一个功能 | **DevTools CLI** + 在设置里启用 `bash` 工具 |

## v0.1 <Badge text="最新" variant="success" />

首个公开版本。

- Chrome 侧边栏对话，带历史记录、模型选择器和推理强度选择器。
- DevTools 里的 "Browy" 面板，支持斜杠命令（`/page`、`/net`、
  `/console`、`/perf`、`/eval`）。
- 本地原生消息主机包装 GitHub Copilot CLI。自备 Copilot 订阅。
- 默认开启 30 多种浏览器工具：截图、点击、输入、导航、标签页、
  `evaluate_js`、`run_script`、网络与控制台日志、沙箱磁盘、
  持久化记忆、`await_user`。
- 可选的主机端工具（`bash`、`read_file`、`write_file`、`grep`、
  `glob`、`web_fetch`），默认关闭，按工具粒度在
  设置 → 工具中开启。
- 多浏览器支持：Chrome、Edge、Brave（以及其他基于 Chromium 的浏览器）。
- Windows 一行命令的 PowerShell 安装脚本。

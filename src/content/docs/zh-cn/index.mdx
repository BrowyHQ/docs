## Browy 是什么

Browy 是一个 Chrome 侧边栏智能体。你和它对话，它读取当前标签页内容，
点击页面元素，填写表单，执行 JavaScript，截图，并在多个会话之间记住
上下文。

它由两部分组成：**浏览器扩展**（在
[Chrome Web Store 上](https://chromewebstore.google.com/detail/iondecjdokngnlkfpipgolgkfegpmjca)）
和一个用一行命令安装的**本地原生消息主机**。
[安装指南](/zh-cn/install/)涵盖了两部分。

## 为什么用 Browy

<CardGrid stagger>
  <Card title="在浏览器里运行" icon="laptop">
    Chrome 侧边栏加一个小的本地辅助进程。直接操作你已经打开的标签页，
    保留你的 cookie 和登录态。
  </Card>
  <Card title="自由切换模型" icon="setting">
    随时在前沿模型之间切换（Claude、GPT、Gemini、Llama），
    用 `/model` 命令或在设置里更改。
  </Card>
  <Card title="基于可访问性树工作" icon="seti:html">
    智能体使用索引化的可访问性节点而非 CSS 选择器，所以不会因为
    类名变化而失效。需要时也可以让它执行任意 JavaScript。
  </Card>
  <Card title="开源" icon="open-book">
    Apache 2.0 许可证。阅读智能体循环、fork 一份、自己构建都可以。
    浏览器工具可以在设置里单独关闭；主机的 shell 和文件系统默认是
    关闭的，需要逐个工具显式打开。
  </Card>
</CardGrid>

## 开始使用

<Aside type="tip" title="两步安装">
**1.** [在 Chrome Web Store 添加 Browy](https://chromewebstore.google.com/detail/iondecjdokngnlkfpipgolgkfegpmjca)（一键安装）。

**2.** 安装本地原生消息主机。Windows 用户打开 PowerShell：

```powershell
irm https://github.com/BrowyHQ/browy/releases/latest/download/install.ps1 | iex
```

macOS / Linux：

```bash
curl -fsSL https://github.com/BrowyHQ/browy/releases/latest/download/install.sh | bash
```

从中国大陆访问 GitHub 偶尔会慢，参见
[中国开发者指南](/zh-cn/china-setup/)中的镜像方案。
</Aside>

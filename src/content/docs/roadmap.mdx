hand-curated roadmap page that drifts out of date, the live state of
the project lives on GitHub — same place the work happens.

<Aside type="tip" title="Want to influence priorities?">
The fastest way to push something up the queue is to **open an issue
with a clear use case**, or react with 👍 on an existing one. We watch
both.
</Aside>

## Contributing

Contributions are welcome. The repo's
[`CONTRIBUTING.md`](https://github.com/BrowyHQ/browy/blob/main/CONTRIBUTING.md)
covers the setup. Good first issues are tagged `good first issue` on
the [issue tracker](https://github.com/BrowyHQ/browy/labels/good%20first%20issue).

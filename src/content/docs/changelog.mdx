## v0.1 <Badge text="Latest" variant="success" />

First public release.

- Chrome side-panel chat with history, model picker, and
  reasoning-effort selector.
- DevTools "Browy" panel with slash commands (`/page`, `/net`,
  `/console`, `/perf`, `/eval`).
- Native messaging host wrapping the GitHub Copilot CLI — bring your
  own Copilot subscription, no Browy server.
- 30+ browser tools by default: snapshot, click, type, navigate, tabs,
  `evaluate_js`, `run_script`, network/console logs, sandboxed disk,
  persistent memory, `await_user`.
- Optional host tools (`bash`, `read_file`, `write_file`, `grep`,
  `glob`, `web_fetch`) — off by default, enable per-tool under
  Settings → Tools.
- Multi-browser support: Chrome, Edge, Brave (and other Chromium
  builds).
- One-line PowerShell installer for Windows.

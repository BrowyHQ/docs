## What Browy is

Browy is a Chrome side-panel agent. You chat with it; it reads the active
tab, clicks through pages, fills forms, runs JavaScript, captures
screenshots, and remembers things across sessions.

It ships in two parts: the **browser extension** (on the
[Chrome Web Store](https://chromewebstore.google.com/detail/iondecjdokngnlkfpipgolgkfegpmjca))
and a small **native messaging host** you install with one command. The
[install guide](/install/) covers both.

## Why Browy

<CardGrid stagger>
  <Card title="Lives in the browser" icon="laptop">
    A Chrome side panel and a small native helper. Works against the
    tabs you already have open.
  </Card>
  <Card title="Pick your model" icon="setting">
    Switch between frontier models (Claude, GPT, Gemini, Llama) on
    the fly with `/model` or in Settings.
  </Card>
  <Card title="Works on accessibility tree" icon="seti:html">
    The agent acts on indexed accessibility nodes instead of CSS selectors,
    so it doesn't break when class names change. JavaScript evaluation is
    available when the snapshot can't express what you need.
  </Card>
  <Card title="Open source" icon="open-book">
    Apache 2.0. Read the agent loop, fork it, build it yourself.
    Browser tools can be turned off individually in Settings; host
    shell and filesystem are off by default and only reachable if you
    opt in per-tool.
  </Card>
</CardGrid>

## Get started

<Aside type="tip" title="Install in two steps">
**1.** [Add Browy from the Chrome Web Store](https://chromewebstore.google.com/detail/iondecjdokngnlkfpipgolgkfegpmjca) (one click).

**2.** Install the native host. On Windows, open PowerShell:

```powershell
irm https://github.com/BrowyHQ/browy/releases/latest/download/install.ps1 | iex
```

On macOS / Linux:

```bash
curl -fsSL https://github.com/BrowyHQ/browy/releases/latest/download/install.sh | bash
```
</Aside>

## What Browy is

Browy is a Chrome side-panel agent. You chat with it in plain English; it
sees your active tab, clicks through pages, fills forms, runs JS, captures
screenshots, and remembers things across sessions. It uses **your own GitHub
Copilot subscription** as the LLM — there is no Browy server.

## Why Browy

<CardGrid stagger>
  <Card title="Lives in your browser" icon="laptop">
    A Chrome side panel + a small native helper. No headless instance, no
    second profile, no cloud browser. The agent works on the tabs you
    already have open.
  </Card>
  <Card title="Your Copilot, your machine" icon="seti:github">
    All LLM traffic flows through your GitHub Copilot subscription via a
    local CLI. Browy itself runs no server and collects no telemetry.
  </Card>
  <Card title="Sees pages, not pixels" icon="setting">
    Snapshots the accessibility tree and acts by index — no brittle CSS
    selectors. Falls back to JS evaluation when the snapshot can't
    express what you need.
  </Card>
  <Card title="Open source" icon="open-book">
    Apache 2.0. Audit the agent loop, fork it, build it yourself.
    Power-user tools (host shell, filesystem) are off by default and
    toggle on per-tool in Settings.
  </Card>
</CardGrid>

## Get started

<Aside type="tip" title="Install in one line">
On Windows, open PowerShell and run:

```powershell
irm https://github.com/BrowyHQ/browy/releases/latest/download/install.ps1 | iex
```

Then load the unpacked extension from `~/Downloads/Browy-Extension/` in
`chrome://extensions` (Developer Mode → "Load unpacked").
</Aside>

## Status <Badge text="Early access" variant="caution" />

Browy is in early access. The native host installer ships from
[GitHub Releases](https://github.com/BrowyHQ/browy/releases). The Chrome
Web Store and Edge Add-ons listings are in review — you'll be able to
install with one click once they're approved.
